# iKKNews
React Native project
>> Develop base on React Native 0.54
## How to run
```
$ git clone https://github.com/Banzuofan/iKKNews.git
$ cd iKKNews
$ npm install
$ react-native run-ios/run-android
```

## Usage
1. redux
2. redux-actions
3. react-navigation
4. FlatList, pull dowm to refresh & pull up to load more data.
